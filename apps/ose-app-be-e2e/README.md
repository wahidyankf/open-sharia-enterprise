# ose-app-be-e2e

Playwright-BDD backend E2E tests for ose-app-be.

## Quick Start

1. Start BE: `nx dev ose-app-be`
2. Run E2E: `nx run ose-app-be-e2e:test:e2e`

## Commands

| Command                               | Description                 |
| ------------------------------------- | --------------------------- |
| `nx run ose-app-be-e2e:test:e2e`      | Run BE E2E tests headlessly |
| `nx run ose-app-be-e2e:spec-coverage` | Check Gherkin step coverage |
