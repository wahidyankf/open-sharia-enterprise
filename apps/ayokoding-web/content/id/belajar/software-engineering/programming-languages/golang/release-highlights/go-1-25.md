## Ikhtisar

Go 1.25, dirilis pada Agustus 2025 dengan versi stabil terbaru 1.25.6 (15 Januari 2026), memperkenalkan peningkatan garbage collection eksperimental, API JSON yang dimodernisasi, dan dukungan deployment kontainer yang lebih baik.

## Fitur Utama

### Green Tea GC (Eksperimental)

Garbage collector eksperimental mengurangi overhead GC sebesar 10-40%:

```bash
# Build dengan Green Tea GC diaktifkan
GOEXPERIMENT=greenteagc go build

# Jalankan dengan Green Tea GC
GOEXPERIMENT=greenteagc go run main.go
```

Terbaik untuk program yang berat GC dengan tingkat alokasi tinggi.

### Package encoding/json/v2

Revisi API JSON mayor dengan tiga package baru:

```go
import jsonv2 "encoding/json/v2"

type User struct {
    ID   int    `json:"id"`
    Name string `json:"name"`
}

// Marshal (encode)
user := User{ID: 1, Name: "Alice"}
data, err := jsonv2.Marshal(user)

// Unmarshal (decode)
var decoded User
err = jsonv2.Unmarshal(data, &decoded)
```

**Peningkatan dari v1**:

- Pesan error lebih baik
- API streaming
- Marshaler kustom dengan konteks
- Performa lebih baik

### GOMAXPROCS Sadar Kontainer

Otomatis mendeteksi kuota CPU di lingkungan kontainer:

```yaml
# Deployment Kubernetes
resources:
  limits:
    cpu: "2" # Go 1.25 mendeteksi ini otomatis
  requests:
    cpu: "1"
```

Tidak perlu pengaturan GOMAXPROCS manual:

```go
// Perilaku Go 1.25:
// - Mesin fisik: 64 CPU
// - Batas kontainer: 2 CPU
// - GOMAXPROCS: 2 (otomatis terdeteksi)

func main() {
    fmt.Printf("GOMAXPROCS: %d\n", runtime.GOMAXPROCS(0))
    // Output di kontainer: GOMAXPROCS: 2
}
```
