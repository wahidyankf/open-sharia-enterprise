## Ikhtisar

Go 1.20 memperkenalkan Profile-Guided Optimization (PGO) sebagai fitur pratinjau, bersama dengan peningkatan coverage dan penanganan error yang lebih baik. Dirilis pada Februari 2023, rilis ini berfokus pada performa dan kemampuan pengujian.

## Fitur Utama

### Profile-Guided Optimization (Pratinjau)

Optimalkan build menggunakan profil runtime:

```bash
# Kumpulkan profil
go test -cpuprofile=cpu.prof

# Build dengan PGO
go build -pgo=cpu.prof
```

### Penanganan Error yang Lebih Baik

Gabungkan beberapa error:

```go
err1 := errors.New("error pertama")
err2 := errors.New("error kedua")
combined := errors.Join(err1, err2)
```

### Konversi Slice ke Array

Konversi tanpa salinan:

```go
slice := []byte("hello")
array := [5]byte(slice)  // Konversi didukung
```
