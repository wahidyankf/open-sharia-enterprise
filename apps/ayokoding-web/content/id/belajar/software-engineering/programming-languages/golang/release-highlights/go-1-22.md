## Ikhtisar

Go 1.22, dirilis pada Februari 2024, mengatasi masalah lama dan memodernisasi paket pustaka standar kunci. Rilis ini memperbaiki masalah scoping variabel loop yang sudah ada selama satu dekade, memperkenalkan pola routing HTTP yang powerful, dan menyediakan API generasi angka acak modern.

## Fitur Utama

### Variabel Loop Per-Iterasi

Memperbaiki bug closure dengan membuat variabel baru per iterasi:

```go
// Go 1.22: Setiap goroutine melihat nilai sendiri
for i := 0; i < 3; i++ {
    go func() {
        fmt.Println(i)  // Benar: 0, 1, 2 (atau permutasi)
    }()
}
```

Sebelum Go 1.22, semua goroutine akan mencetak 3.

### Range untuk Integer

Iterasi integer yang disederhanakan:

```go
// Go 1.22: Range untuk integer
for i := range 5 {
    fmt.Println(i)  // 0, 1, 2, 3, 4
}

// Ulangi N kali
for range 10 {
    doSomething()
}
```

### HTTP Routing yang Ditingkatkan

Handler spesifik metode dan wildcard path:

```go
mux := http.NewServeMux()

// Pembatasan metode
mux.HandleFunc("GET /users", listUsers)
mux.HandleFunc("POST /users", createUser)

// Wildcard path
mux.HandleFunc("GET /users/{id}", getUser)
mux.HandleFunc("GET /files/{path...}", serveFile)

// Ekstrak nilai path
func getUser(w http.ResponseWriter, r *http.Request) {
    id := r.PathValue("id")  // Tidak perlu parsing
    // ...
}
```

### Package math/rand/v2

API angka acak yang dimodernisasi:

```go
import "math/rand/v2"

// Otomatis di-seed (tidak perlu seed manual)
n := rand.IntN(100)  // 0 hingga 99
f := rand.Float64()  // 0.0 hingga 1.0
```
