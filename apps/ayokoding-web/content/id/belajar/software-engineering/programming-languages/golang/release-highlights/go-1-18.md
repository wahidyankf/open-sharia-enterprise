## Ikhtisar

Go 1.18, dirilis pada Maret 2022, merupakan salah satu rilis paling signifikan dalam sejarah Go. Rilis ini memperkenalkan tiga fitur utama yang mengubah fundamental cara pengembang Go menulis, menguji, dan mengorganisasi kode:

- **Generics (Parameter Tipe)**: Menulis kode yang dapat digunakan kembali dengan aman tipe tanpa mengorbankan performa
- **Dukungan Fuzzing**: Fuzzing bawaan untuk menemukan bug melalui generasi input otomatis
- **Mode Workspace**: Menyederhanakan pengembangan lintas modul

Rilis ini menghadirkan kemampuan pemrograman modern sambil mempertahankan kesederhanaan dan performa Go.

## Generics (Parameter Tipe)

Generics memungkinkan Anda menulis fungsi dan tipe yang bekerja dengan tipe apapun sambil mempertahankan keamanan tipe.

```go
// Fungsi generic dengan parameter tipe
func Print[T any](value T) {
    fmt.Println(value)
}

// Penggunaan dengan inferensi tipe
Print(42)          // int
Print("hello")     // string
```

## Dukungan Fuzzing

Fuzzing natif secara otomatis menemukan bug melalui generasi input acak:

```go
func FuzzReverse(f *testing.F) {
    // Seed corpus
    f.Add("hello")
    f.Add("world")

    // Target fuzz
    f.Fuzz(func(t *testing.T, input string) {
        reversed := Reverse(input)
        doubleReversed := Reverse(reversed)

        if input != doubleReversed {
            t.Errorf("Reverse(Reverse(%q)) = %q, ingin %q",
                input, doubleReversed, input)
        }
    })
}
```

## Mode Workspace

Mode workspace menyederhanakan pengembangan multi-modul tanpa direktif `replace` yang kompleks:

```bash
# Inisialisasi workspace dengan modul
go work init ./app ./lib
```
