## Ikhtisar

Go 1.23, dirilis pada Agustus 2024, memperkenalkan fungsi iterator yang memungkinkan pola iterasi kustom dengan loop for-range, package unique untuk deduplikasi nilai yang efisien, dan peningkatan perilaku timer yang penting.

## Fitur Utama

### Fungsi Iterator

Range untuk pola iterasi kustom:

```go
// Fungsi iterator
func Count(n int) func(func(int) bool) {
    return func(yield func(int) bool) {
        for i := 0; i < n; i++ {
            if !yield(i) {
                return
            }
        }
    }
}

// Range untuk iterator
for i := range Count(5) {
    fmt.Println(i)  // 0, 1, 2, 3, 4
}
```

Menggunakan tipe `iter.Seq`:

```go
import "iter"

func Fibonacci(max int) iter.Seq[int] {
    return func(yield func(int) bool) {
        a, b := 0, 1
        for a <= max {
            if !yield(a) {
                return
            }
            a, b = b, a+b
        }
    }
}

for n := range Fibonacci(100) {
    fmt.Println(n)  // 0, 1, 1, 2, 3, 5, 8, 13, ...
}
```

### Package unique

Kanonisasi nilai untuk optimasi memori:

```go
import "unique"

// Kanonisasi nilai
h1 := unique.Make("hello")
h2 := unique.Make("hello")

fmt.Println(h1 == h2)  // true (Handle yang sama, berbagi penyimpanan)
```

### Fungsi Iterator slices dan maps

```go
import "slices"

s := []string{"a", "b", "c"}

// Iterasi dengan indeks
for i, v := range slices.All(s) {
    fmt.Printf("%d: %s\n", i, v)
}

// Hanya nilai
for v := range slices.Values(s) {
    fmt.Println(v)
}
```
