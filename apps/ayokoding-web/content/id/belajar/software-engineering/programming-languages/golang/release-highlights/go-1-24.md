## Ikhtisar

Go 1.24, dirilis pada Februari 2025, memberikan peningkatan performa substansial dan primitif keamanan baru. Implementasi map Swiss Tables menyediakan peningkatan CPU keseluruhan 2-3%, runtime.AddCleanup menawarkan alternatif yang lebih baik dari finalizer, dan os.Root memungkinkan operasi filesystem terisolasi yang aman.

## Fitur Utama

### Implementasi Map Swiss Tables

Operasi map lebih cepat mengurangi overhead CPU keseluruhan sebesar 2-3%:

```go
// Tidak ada perubahan kode diperlukan!
// Semua map otomatis menggunakan Swiss Tables
users := make(map[string]User)
users["alice"] = User{Name: "Alice"}

// Manfaat:
// - Pencarian lebih cepat (15-25%)
// - Penyisipan/pembaruan lebih cepat (10-15%)
// - Penghapusan lebih cepat (10-15%)
```

### runtime.AddCleanup

Mekanisme pembersihan modern menggantikan finalizer:

```go
import "runtime"

type Resource struct {
    id   int
    file *os.File
}

func NewResource(id int, filename string) (*Resource, error) {
    file, err := os.Open(filename)
    if err != nil {
        return nil, err
    }

    r := &Resource{id: id, file: file}

    // Lampirkan fungsi pembersihan
    runtime.AddCleanup(r, func(r *Resource) {
        r.file.Close()
    })

    return r, nil
}
```

### os.Root untuk Operasi Filesystem Terisolasi

Cegah serangan path traversal:

```go
// Buka direktori root untuk operasi terisolasi
root, err := os.OpenRoot("/var/app/data")
if err != nil {
    log.Fatal(err)
}
defer root.Close()

// Buka file (relatif terhadap root)
file, err := root.Open("config.json")

// Coba path traversal
file, err = root.Open("../../etc/passwd")
// Error: Path melarikan diri dari direktori root
```

### Alias Tipe Generik

Dukungan penuh untuk alias tipe berparameter:

```go
// Alias tipe generik
type Pair[T any] = struct {
    First  T
    Second T
}

p := Pair[int]{First: 1, Second: 2}
```
