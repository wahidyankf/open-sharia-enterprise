## Ikhtisar

Go 1.21, dirilis pada Agustus 2023, menjadikan Profile-Guided Optimization siap produksi dan memperkenalkan fungsi bawaan yang sering diminta. Rilis ini berfokus pada peningkatan performa dan pengalaman pengembang.

## Fitur Utama

### PGO Siap Produksi

Profile-Guided Optimization memberikan peningkatan performa 2-7%:

```bash
# Kumpulkan profil
curl -o cpu.prof http://localhost:6060/debug/pprof/profile?seconds=30

# Tempatkan profil
cp cpu.prof cmd/myapp/default.pgo

# Build (PGO otomatis diaktifkan)
go build ./cmd/myapp
```

### Fungsi Bawaan Baru

**min dan max**:

```go
a := min(1, 2, 3)      // 1
b := max(5.5, 2.3)     // 5.5
c := min("apple", "banana")  // "apple"
```

**clear**:

```go
// Hapus map
m := map[string]int{"a": 1, "b": 2}
clear(m)  // Penghapusan semua entri yang efisien

// Nolkan elemen slice
s := []int{1, 2, 3}
clear(s)  // [0 0 0]
```

### Manajemen Toolchain Go

Tentukan persyaratan versi Go di go.mod:

```go
module example.com/myapp

go 1.21
toolchain go1.21.5  // Versi toolchain spesifik
```
