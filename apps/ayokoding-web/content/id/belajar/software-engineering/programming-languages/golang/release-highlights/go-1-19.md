## Ikhtisar

Go 1.19 memperkenalkan peningkatan performa, manajemen memori yang lebih baik, dan peningkatan tooling. Dirilis pada Agustus 2022, rilis ini melanjutkan evolusi Go menuju performa dan pengalaman pengembang yang lebih baik.

## Fitur Utama

### Batas Memori Lunak (GOMEMLIMIT)

Mengontrol penggunaan memori heap maksimum:

```bash
# Atur batas memori 2GB
GOMEMLIMIT=2GiB go run main.go
```

### Garbage Collector Lebih Cepat

GC yang lebih baik dengan latensi lebih rendah dan throughput lebih tinggi.

### Tipe atomic.Pointer

Pointer atomik yang aman tipe:

```go
var ptr atomic.Pointer[Config]

config := &Config{Host: "localhost"}
ptr.Store(config)

current := ptr.Load()
```
