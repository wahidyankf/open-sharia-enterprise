Clojure tutorials offer multiple learning paths designed for different experience levels and learning styles.

## Learning Paths

**By Example** - Code-first approach with 80 annotated, runnable examples covering 95% of production Clojure. Best for experienced developers who prefer learning through code exploration.

## Choosing Your Path

**New to Clojure?** Start with Initial Setup, then Quick Start to write your first program in minutes.

**Experienced Developer?** Jump to By Example for rapid exploration through comprehensive code samples.

**Building Production Systems?** Follow By Concept to understand Clojure's philosophy and patterns deeply.
