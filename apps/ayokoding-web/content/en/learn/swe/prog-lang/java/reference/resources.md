## Official Documentation

### Oracle Java Documentation

**Java SE Documentation**

- **URL**: https://docs.oracle.com/en/java/javase/
- **Description**: Official Java Standard Edition documentation covering all Java versions
- **Best for**: API references, language specifications, tutorials
- **Latest LTS**: Java 21 (September 2023)

**Java Language Specification**

- **URL**: https://docs.oracle.com/javase/specs/
- **Description**: Definitive technical specification of the Java language
- **Best for**: Understanding language semantics, edge cases, formal definitions

**Java API Documentation**

- **URL**: https://docs.oracle.com/en/java/javase/21/docs/api/
- **Description**: Complete API reference for Java SE classes and interfaces
- **Best for**: Method signatures, class hierarchies, usage examples

### OpenJDK

**OpenJDK Project**

- **URL**: https://openjdk.org/
- **Description**: Open-source reference implementation of Java SE
- **Best for**: Source code, JEPs, community development

**JDK Enhancement Proposals (JEPs)**

- **URL**: https://openjdk.org/jeps/0
- **Description**: Proposals for enhancements to the JDK
- **Notable JEPs**:
  - JEP 395: Records (Java 16)
  - JEP 409: Sealed Classes (Java 17)
  - JEP 394: Pattern Matching for instanceof (Java 16)
  - JEP 440: Record Patterns (Java 21)
  - JEP 444: Virtual Threads (Java 21)

## Books

### Fundamentals

**Effective Java (3rd Edition)**

- **Author**: Joshua Bloch
- **Publisher**: Addison-Wesley (2017)
- **Description**: 90 best practices for writing high-quality Java code
- **Topics**: Object creation, methods, classes, generics, enums, concurrency
- **Level**: Intermediate to Advanced
- **Why Read**: Industry-standard best practices, must-read for Java developers

**Head First Java (3rd Edition)**

- **Authors**: Kathy Sierra, Bert Bates, Trisha Gee
- **Publisher**: O'Reilly Media (2022)
- **Description**: Visual, engaging introduction to Java fundamentals
- **Topics**: OOP, inheritance, polymorphism, exceptions, collections, GUI
- **Level**: Beginner
- **Why Read**: Accessible learning style, solid foundation

**Core Java Volume I - Fundamentals (12th Edition)**

- **Authors**: Cay S. Horstmann
- **Publisher**: Pearson (2022)
- **Description**: Comprehensive guide to Java SE fundamentals
- **Topics**: Objects, classes, inheritance, interfaces, lambda expressions, Collections API
- **Level**: Beginner to Intermediate
- **Why Read**: Thorough coverage, authoritative reference

### Advanced Topics

**Java Concurrency in Practice**

- **Authors**: Brian Goetz, Tim Peierls, et al.
- **Publisher**: Addison-Wesley (2006)
- **Description**: Definitive guide to writing concurrent Java programs
- **Topics**: Thread safety, synchronization, concurrent collections, executors
- **Level**: Advanced
- **Why Read**: Essential for concurrent programming, written by Java concurrency experts

**Thinking in Java (4th Edition)**

- **Author**: Bruce Eckel
- **Publisher**: Prentice Hall (2006)
- **Description**: In-depth exploration of Java concepts and philosophy
- **Topics**: OOP principles, generics, annotations, I/O, concurrency
- **Level**: Intermediate
- **Why Read**: Deep conceptual understanding, thought-provoking explanations

**Modern Java in Action (2nd Edition)**

- **Authors**: Raoul-Gabriel Urma, Mario Fusco, Alan Mycroft
- **Publisher**: Manning (2018)
- **Description**: Java 8+ features including lambdas, streams, and reactive programming
- **Topics**: Lambda expressions, Stream API, CompletableFuture, modules
- **Level**: Intermediate to Advanced
- **Why Read**: Modern Java practices, functional programming in Java

### Enterprise and Frameworks

**Spring in Action (6th Edition)**

- **Author**: Craig Walls
- **Publisher**: Manning (2022)
- **Description**: Comprehensive guide to Spring Framework
- **Topics**: Spring Boot, Spring MVC, Spring Data, Spring Security, microservices
- **Level**: Intermediate to Advanced
- **Why Read**: Industry-standard enterprise framework

**Jakarta EE Cookbook (2nd Edition)**

- **Authors**: Elder Moraes
- **Publisher**: Packt (2020)
- **Description**: Practical recipes for Jakarta EE development
- **Topics**: CDI, JPA, JAX-RS, WebSocket, JSON-B
- **Level**: Intermediate
- **Why Read**: Enterprise Java patterns, real-world solutions

### Design and Architecture

**Clean Code**

- **Author**: Robert C. Martin
- **Publisher**: Prentice Hall (2008)
- **Description**: Best practices for writing maintainable code
- **Topics**: Naming, functions, comments, formatting, error handling
- **Level**: All levels
- **Why Read**: Industry-standard code quality principles (language-agnostic but Java examples)

**Design Patterns: Elements of Reusable Object-Oriented Software**

- **Authors**: Gang of Four (Erich Gamma, et al.)
- **Publisher**: Addison-Wesley (1994)
- **Description**: Classic catalog of 23 design patterns
- **Topics**: Creational, structural, behavioral patterns
- **Level**: Intermediate to Advanced
- **Why Read**: Foundational design patterns, reusable solutions

## Learning Paths

### Backend Developer Path

1. **Foundation**: Core Java (syntax, OOP, collections, exceptions)
2. **Database**: SQL, JDBC, JPA/Hibernate
3. **Web**: Servlets, Spring Boot, REST APIs
4. **Testing**: JUnit, Mockito, integration testing
5. **Tools**: Maven/Gradle, Git, Docker
6. **Advanced**: Microservices, Spring Cloud, Kubernetes

### Android Developer Path

1. **Foundation**: Core Java, Kotlin basics
2. **Android**: Activities, Fragments, Layouts
3. **Architecture**: MVVM, Clean Architecture
4. **Libraries**: Retrofit, Room, Dagger/Hilt
5. **Testing**: JUnit, Espresso, Robolectric
6. **Advanced**: Jetpack Compose, Coroutines, Flow

### Big Data/Analytics Path

1. **Foundation**: Core Java, collections, concurrency
2. **Big Data**: Hadoop, Spark, Kafka
3. **Database**: NoSQL (MongoDB, Cassandra), SQL
4. **Processing**: Stream processing, batch processing
5. **Tools**: Maven/Gradle, Docker, cloud platforms
6. **Advanced**: Machine learning libraries, distributed systems
