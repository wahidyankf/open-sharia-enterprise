# Web Development Tools and Frameworks

This section contains comprehensive tutorials and guides for web development frameworks and tools, organized by technology stack.

## Available Frameworks

### Frontend Frameworks

- **[React](/en/learn/software-engineering/platform-web/tools/fe-react)** - Component-based JavaScript library for building user interfaces
- **[Next.js](/en/learn/software-engineering/platform-web/tools/fe-nextjs)** - React framework with server-side rendering and full-stack capabilities

### Backend Frameworks

- **[Elixir Phoenix](/en/learn/software-engineering/platform-web/tools/elixir-phoenix)** - High-performance web framework built on the Erlang VM
- **[Phoenix LiveView](/en/learn/software-engineering/platform-web/tools/elixir-phoenix-liveview)** - Real-time, server-rendered web applications using Phoenix
- **[Spring Framework](/en/learn/software-engineering/platform-web/tools/jvm-spring)** - Comprehensive Java framework for enterprise applications
- **[Spring Boot](/en/learn/software-engineering/platform-web/tools/jvm-spring-boot)** - Opinionated Spring configuration for rapid application development

## Learning Structure

Each framework follows a consistent tutorial structure:

1. **Overview** - Introduction, key features, when to use
2. **Initial Setup** - Installation and project setup (0-5% coverage)
3. **Quick Start** - Complete working example (5-30% coverage)
4. **By Example** - 75-90 annotated code examples (95% coverage)

Choose a framework based on your project requirements, team expertise, and technical constraints.
