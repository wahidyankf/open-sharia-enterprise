# Python Documentation Templates

This directory contains reusable code templates and patterns for Python development.

## Purpose

Templates provide standardized starting points for common Python patterns,
reducing repetition and ensuring consistency across documentation and code examples.

## Template Naming Convention

Templates follow the pattern: `ex-soen-prla-py-te__[pattern-name].md`

Where:

- `ex-soen-prla-py` - Explanation, Software Engineering, Programming Language, Python
- `te` - Template indicator
- `[pattern-name]` - Descriptive name of the pattern (e.g., `fastapi-endpoint`)

## Templates

This directory is currently a placeholder. Templates will be added as common patterns are identified.

Suggested starting template: `ex-soen-prla-py-te__fastapi-endpoint.md`

## Usage

Templates in this directory can be used as:

- Starting points for new implementations
- Reference patterns for code review
- Educational examples in documentation
