# TypeScript Documentation Templates

This directory contains reusable code templates and patterns for TypeScript development.

## Purpose

Templates provide standardized starting points for common TypeScript patterns,
reducing repetition and ensuring consistency across documentation and code examples.

## Template Naming Convention

Templates follow the pattern: `ex-soen-prla-ty-te__[pattern-name].md`

Where:

- `ex-soen-prla-ty` - Explanation, Software Engineering, Programming Language, TypeScript
- `te` - Template indicator
- `[pattern-name]` - Descriptive name of the pattern (e.g., `service-class`)

## Templates

This directory is currently a placeholder. Templates will be added as common patterns are identified.

Suggested starting template: `ex-soen-prla-ty-te__service-class.md`

## Usage

Templates in this directory can be used as:

- Starting points for new implementations
- Reference patterns for code review
- Educational examples in documentation
