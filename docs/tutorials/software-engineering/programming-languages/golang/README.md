# Go (Golang) Tutorials

This directory contains learning-oriented tutorials focused on the Go programming language (Golang), covering fundamentals, best practices, and practical applications.

## Available Tutorials

- [**Golang Crash Course**](./tu-soen-prla-gola__crash-course.md) - Learn Go language mechanics from fundamentals to advanced patterns (⏱️ 2-3 hours)
- [**Golang Cookbook**](./tu-soen-prla-gola__cookbook.md) - Practical Go patterns and recipes for real-world problems (⏱️ 3-4 hours)

## Tutorial Structure

Each tutorial follows the [Diátaxis framework](../../../explanation/conventions/ex-co__diataxis-framework.md) principles for learning-oriented content:

- **Learning-oriented**: Designed to help newcomers learn Go by doing
- **Step-by-step**: Clear, sequential instructions with progressive complexity
- **Practical**: Hands-on exercises with working Go code examples
- **Achievable**: Complete, functional examples that build confidence

## Topics Covered

Go tutorials may include:

- Go fundamentals (syntax, types, variables, functions)
- Concurrency and goroutines
- Channels and synchronization
- Interfaces and structs
- Error handling patterns
- Standard library usage
- Testing and benchmarking
- Web development with Go
- Building CLI applications
- Microservices and APIs
